O que é Design Responsivo?

Design Responsivo é a capacidade de um website de se ajustar e adaptar a diferentes tamanhos de tela e dispositivos, desde celulares pequenos a monitores de desktop gigantes. O objetivo é proporcionar uma boa experiência de usuário para todos.

A base para um site responsivo começa no HTML, com a inclusão da meta tag viewport no <head>. 

<!-- Esta tag instrui o navegador a ajustar a largura da página à largura da tela do dispositivo e a definir o nível de zoom inicial. -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Sem esta tag, os navegadores de dispositivos móveis tentariam renderizar a página como se fosse uma tela de desktop, resultando em uma página "diminuída" e difícil de ler.

Media Queries: O Cérebro da Responsividade

Media Queries são a principal ferramenta do CSS para aplicar estilos condicionalmente. Elas permitem que você crie regras de estilo que só são ativadas quando certas condições são satisfeitas, como a largura da tela.

Sintaxe

@media MediaType and (MediaCondition) {
    /* Código CSS que será aplicado se a condição for verdadeira */
    body {
        background-color: lightblue;
    }
}
  • MediaType (opcional): Define o tipo de mídia. Os mais comuns são screen (telas), print (para impressão) e all (padrão).
  • MediaCondition: A condição a ser satisfeita. As mais usadas para responsividade são min-width (largura mínima) e max-width (largura máxima).

Introdução ao Flexbox

O Flexible Box Layout, ou Flexbox, é um modelo de layout unidimensional projetado para nos dar um controle poderoso sobre o alinhamento, espaçamento e ordenação dos itens dentro de um container.

Para começar a usar, basta aplicar display: flex a um elemento pai (o flex container). Seus filhos diretos se tornam automaticamente flex items.

Eixo Principal (Main Axis) e Eixo Transversal (Cross Axis)

O Flexbox trabalha com dois eixos. Entendê-los é crucial:

  • O Eixo Principal é o eixo primário ao longo do qual os itens são dispostos. Por padrão, ele é horizontal (flex-direction: row).
  • O Eixo Transversal é perpendicular ao eixo principal. Por padrão, é vertical.

Propriedades do Flex Container

Estas são as propriedades que você aplica ao elemento pai para controlar o layout geral dos seus filhos.

flex-direction

Altera a direção do eixo principal (e, consequentemente, do eixo transversal).

1
2
3
.container { display: flex; flex-direction: row; }

justify-content

Alinha os itens ao longo do eixo principal. É a propriedade mais usada para espaçamento.

1
2
3
.container { display: flex; justify-content: flex-start; }

align-items

Alinha os itens ao longo do eixo transversal (dentro de uma única linha).

1
2
3 
.container { display: flex; align-items: stretch; }

flex-wrap

Por padrão, os itens tentam caber em uma única linha. Esta propriedade permite que eles quebrem para a próxima linha.

1
2
3
4
5
6
.container { display: flex; flex-wrap: nowrap; }

align-content

Similar ao justify-content, mas alinha as linhas do container no eixo transversal. Só tem efeito quando há múltiplas linhas (flex-wrap: wrap).

1
2
3
4
5
6
.container { display: flex; flex-wrap: wrap; align-content: stretch; }

gap

A maneira moderna e simples de adicionar espaçamento entre os itens flex, tanto nas colunas quanto nas linhas. Substitui a necessidade de margens complexas. 

.container {
    display: flex;
    gap: 10px; /* 10px entre linhas e colunas */
    gap: 20px 10px; /* 20px entre linhas, 10px entre colunas */
}

Propriedades dos Itens Flex

Estas propriedades são aplicadas diretamente aos elementos filhos (os flex items) para dar-lhes um comportamento individual dentro do container.

align-self

Permite sobrescrever o align-items do container para um único item.

Padrão
align-self: center
align-self: flex-end

flex-grow, flex-shrink e flex-basis

Este trio poderoso controla a flexibilidade dos itens.

  • flex-grow (padrão: 0): Define a capacidade de um item "crescer" e ocupar o espaço livre no container. Um valor de 1 em todos os itens faz com que eles ocupem o espaço livre igualmente.
  • flex-shrink (padrão: 1): Define a capacidade de um item "encolher" se não houver espaço suficiente.
  • flex-basis (padrão: auto): Define o tamanho inicial de um item antes que o espaço livre seja distribuído.

Geralmente, essas três propriedades são definidas usando o atalho flex. 

.item {
    /* flex: <flex-grow> <flex-shrink> <flex-basis>; */
    flex: 1 1 auto;
}
