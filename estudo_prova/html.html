O que é HTML?

HTML, ou HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a espinha dorsal de toda a web. Pense nela como o esqueleto de uma página: ela define a estrutura fundamental, organizando o conteúdo em parágrafos, títulos, listas, imagens e muito mais.

É importante ressaltar que HTML não é uma linguagem de programação. Ela não executa lógica ou algoritmos. Em vez disso, é uma linguagem de marcação, que usa "tags" para descrever e formatar o conteúdo para que um navegador saiba como exibi-lo corretamente.

Estrutura Básica de um Documento

Toda página HTML segue uma estrutura padrão, conhecida como boilerplate. Ela contém os elementos essenciais para que o navegador entenda e renderize a página corretamente.

Estrutura Essencial (Boilerplate)
<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título da Página</title>
    </head>
    <body>
        <!-- Todo o conteúdo visível da sua página vai aqui -->
    </body>
</html>

Anatomia da Estrutura:

  • <!DOCTYPE html>: Declaração que define o documento como sendo HTML5. É sempre a primeira linha.
  • <html lang="pt-BR">: O elemento raiz da página. O atributo lang é importante para acessibilidade e SEO, indicando o idioma principal do conteúdo.
  • <head>: Contém "metadados" sobre a página – informações que não são exibidas diretamente, como o título, codificação de caracteres e links para arquivos CSS.
  • <meta charset="UTF-8">: Define o conjunto de caracteres para UTF-8, o padrão da web, que suporta acentos e símbolos.
  • <meta name="viewport" ...>: Tag crucial para o design responsivo, instruindo o navegador a ajustar a página à largura da tela do dispositivo.
  • <title>: Define o texto que aparece na aba ou na barra de título do navegador.
  • <body>: Onde todo o conteúdo visível da página (textos, imagens, links, etc.) é colocado.

Elementos de Texto

Estes são os blocos de construção mais comuns para exibir conteúdo textual.

Títulos e Parágrafos

Use os títulos de <h1> a <h6> para criar uma hierarquia de informação na sua página. O <p> é usado para parágrafos de texto.

Ênfase e Importância

É importante diferenciar entre estilo visual e significado semântico.

  • <strong>: Indica que o texto tem grande importância, seriedade ou urgência. Navegadores o exibem em negrito por padrão.
  • <em>: Representa ênfase, como uma palavra que você pronunciaria com mais destaque. Navegadores o exibem em itálico.
  • <b> e <i>: Usados para chamar atenção visualmente (negrito e itálico) sem adicionar importância semântica. Prefira <strong> e <em> sempre que houver um significado por trás do estilo.

Links e Imagens

A web é feita de conexões. Links (ou âncoras) e imagens são fundamentais para isso.

Hiperlinks: A Tag <a>

A tag <a> (âncora) cria hiperlinks. O atributo href especifica o destino do link.

Exemplo de Links
  • Link externo para o Google (abre em nova aba)
  • Link interno para a seção de formulários
  • Enviar um e-mail
<a href="https://www.google.com" target="_blank">Link externo</a>
<a href="#formularios">Link interno</a>
<a href="mailto:contato@exemplo.com">Enviar um e-mail</a>

Imagens: A Tag <img>

A tag <img> é usada para incorporar imagens. É uma tag "vazia", ou seja, não possui conteúdo de fechamento.

  • src: O caminho (URL) para a imagem. É obrigatório.
  • alt: Texto alternativo. Descreve a imagem para leitores de tela e caso a imagem não carregue. É crucial para a acessibilidade.
Exemplo de Imagem com Legenda
<figure>
    <img src="caminho/para/imagem.jpg" alt="Descrição clara da imagem">
    <figcaption>Legenda opcional para a imagem.</figcaption>
</figure>

Tabelas

Tabelas são usadas para exibir dados tabulares de forma organizada em linhas e colunas.

Exemplo de Tabela Completa
Vendas Trimestrais por Região
Região Q1 Q2
Sudeste R$ 150.000 R$ 180.000
Nordeste R$ 90.000 R$ 110.000
Total R$ 240.000 R$ 290.000 
<table>
    <caption>Vendas Trimestrais</caption>
    <thead>
        <tr>
            <th scope="col">Região</th>
            <th scope="col">Q1</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope="row">Sudeste</th>
            <td>R$ 150.000</td>
        </tr>
    </tbody>
</table>

HTML Semântico

HTML semântico significa usar as tags HTML para o seu propósito correto, descrevendo o significado do conteúdo, e não apenas sua aparência. Isso é vital para acessibilidade (leitores de tela), SEO (motores de busca) e manutenção do código.

Em vez de usar <div> para tudo, use tags que descrevam cada parte da sua página.

  • <header>: O cabeçalho da página ou de uma seção, geralmente contendo o logo e a navegação principal.
  • <nav>: Contém os principais links de navegação.
  • <main>: Onde o conteúdo principal e único da página deve estar. Só deve haver um por página.
  • <article>: Para conteúdo independente e autocontido, como um post de blog ou uma notícia.
  • <section>: Um agrupamento temático de conteúdo.
  • <aside>: Para conteúdo relacionado, mas lateral, como uma barra de links ou uma biografia do autor.
  • <footer>: O rodapé da página, com informações de copyright, links de contato, etc.
