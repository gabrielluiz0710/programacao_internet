Introdução ao CSS

CSS, ou Folha de Estilo em Cascata, é a linguagem que usamos para estilizar um documento HTML. Ela descreve como os elementos devem ser renderizados na tela, no papel ou em outras mídias.

Como Inserir CSS

Existem três formas principais de adicionar CSS a uma página:

  1. Embutido (Inline): Usando o atributo style diretamente em uma tag HTML. Não é recomendado por misturar estrutura e apresentação.
  2. Interno: Usando a tag <style> dentro do <head> do documento HTML. Útil para estilos específicos de uma única página.
  3. Externo: Criando um arquivo .css separado e lincando-o no <head>. Esta é a melhor prática, pois permite reutilizar estilos em várias páginas e mantém o código organizado. 
<!-- Exemplo de folha de estilo externa -->
<link rel="stylesheet" href="style.css">

Estrutura de uma Regra CSS

Uma regra CSS é composta por um seletor e um bloco de declaração. Cada declaração inclui uma propriedade e um valor. 

p {
    color: blue; /* Propriedade: color, Valor: blue */
    font-size: 16px;
}

Seletores

Seletores definem quais elementos HTML serão afetados pelas regras de estilo. Dominá-los é fundamental para escrever CSS eficiente.

Seletor de ID: #meu-id
Este elemento tem o ID "titulo-principal" e foi selecionado. 
#titulo-principal {
    background-color: #ffcccc;
}
Seletor de Classe: .minha-classe

Este parágrafo tem a classe "destaque".

Este div também tem a classe "destaque". 
.destaque {
    border: 2px dashed #1dd1a1;
}
Seletor de Descendente: div a

Este é um link descendente dentro da div.

  • Outro link descendente.
Este link não é afetado. 
/* Afeta TODOS os 'a' dentro de '.demo-box' */
.demo-box a {
    color: #1dd1a1;
}

Este parágrafo é um filho direto e foi selecionado.

Este parágrafo NÃO é um filho direto.

/* Afeta apenas o 'p' que é filho direto de '.demo-box' */

}

Pseudo-Classes e Pseudo-Elementos

Pseudo-classes selecionam elementos em um estado especial (ex: :hover), enquanto pseudo-elementos estilizam uma parte específica de um elemento (ex: ::first-line).

Pseudo-Classe: :hover
Passe o mouse sobre mim!
.hover-box:hover {
    background-color: #1dd1a1;
    transform: scale(1.05);
}
Pseudo-Elementos: ::before e ::after

Isto é uma citação importante.

.quote-box::before {
    content: '“';
    font-size: 3rem;
}

Outras pseudo-classes úteis incluem :focus, :checked, :first-child, :nth-child(n), :has() e :is().

Propriedades de Texto

O CSS oferece controle total sobre a aparência do texto.

  • font-family: Define a família da fonte (ex: 'Poppins', 'Arial', sans-serif).
  • font-size: Controla o tamanho do texto.
  • font-weight: Define a "grossura" da fonte (ex: normal, bold, 700).
  • line-height: Controla o espaçamento entre as linhas.
  • color: Define a cor do texto.
  • text-align: Alinha o texto horizontalmente (left, center, right, justify).
  • text-decoration: Adiciona decorações como underline.
  • text-transform: Transforma o texto (uppercase, lowercase, capitalize).

Box Model

Todo elemento em HTML é uma caixa retangular. O Box Model é o conjunto de regras que define como essa caixa é dimensionada e como ela interage com outras caixas. Ele é composto por: Conteúdo, Padding, Borda e Margem.

Margem
Conteúdo + Padding + Borda

Box-Sizing

A propriedade box-sizing altera como a largura (width) e altura (height) totais de um elemento são calculadas.

  • content-box (padrão): width e height se aplicam apenas ao conteúdo. Padding, borda e margem são adicionados a isso.
  • border-box: width e height incluem o conteúdo, o padding e a borda. É muito mais intuitivo para criar layouts.
content-box vs border-box

content-box (Largura total: 200px + 40px padding + 20px borda = 260px)

border-box (Largura total: 200px)

Bordas e Arredondamento

A propriedade border é um atalho para border-width, border-style e border-color. Com border-radius, podemos arredondar os cantos.

border-radius
.radius-box {
    border-radius: 20px;
}

Display e Posicionamento

Essas propriedades controlam como um elemento se comporta no fluxo do documento e onde ele é posicionado.

Display

block, inline e inline-block
display: block (ocupa a largura toda) display: inline (ocupa só o necessário) outro inline. display: inline-block (inline com width/height)

Position

Controla como um elemento é posicionado. Os valores mais comuns são static (padrão), relative, absolute, fixed e sticky.

Exemplos de position
relative
absolute
fixed

(O container tem position: relative, por isso o box 'absolute' se posiciona em relação a ele. O box 'fixed' se posiciona em relação à janela do navegador.)

Transform

A propriedade transform permite mover, rotacionar, escalonar e inclinar elementos.

transform:hover
Passe o mouse
.transform-box:hover {
    transform: rotate(45deg) scale(1.2) translateX(30px);
}

Outras Propriedades Importantes

  • background: Atalho para definir background-color, background-image, background-repeat, background-position, e background-size.
  • box-shadow: Adiciona uma sombra à caixa de um elemento.
  • visibility: Controla a visibilidade de um elemento. visibility: hidden oculta o elemento, mas ele ainda ocupa espaço no layout.
  • overflow: Define o que acontece se o conteúdo de um elemento for maior que sua caixa.

Cascata, Especificidade e Herança

O "C" em CSS significa "Cascata". Isso se refere à forma como os navegadores resolvem conflitos quando múltiplas regras de estilo se aplicam a um mesmo elemento.

  • Cascata: A ordem das regras importa. Se duas regras tiverem a mesma especificidade, a última declarada no CSS vence.
  • Especificidade: Regras mais específicas têm maior peso. Um seletor de ID (#meu-id) é mais específico que um seletor de classe (.minha-classe), que por sua vez é mais específico que um seletor de elemento (p). Estilos inline (style="...") são os mais específicos de todos.
  • Herança: Alguns estilos (como color e font-family) são herdados dos elementos-pai para os elementos-filho. Outros (como border e padding) não são.
